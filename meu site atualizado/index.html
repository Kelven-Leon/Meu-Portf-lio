PraPets

Aplicativo móvel e site desenvolvidos para gestão completa de cuidados para pets, integrando APIs de geolocalização (Google Maps) para exibir clínicas veterinárias próximas ao usuário. O sistema oferece agendamento com notificações push, cadastro e acompanhamento de pets, histórico de vacinas, banhos e tosas, além de suporte a perfis de ONGs e adoção de animais. O design das interfaces foi criado no Figma, priorizando usabilidade e responsividade. O front-end do site foi implementado utilizando HTML, CSS e JavaScript, com uma arquitetura de componentes reutilizáveis para garantir escalabilidade e facilidade de manutenção.
